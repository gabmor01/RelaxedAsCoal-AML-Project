This project explores the reconstruction of visual stimuli from fMRI data using the Natural Scenes Dataset (NSD). Due to computational constraints, we implemented a simplified version of the MindEye architecture, focusing on mapping 15,724 fMRI voxels into the 1,280-dimensional Kandinsky CLIP latent space using a funnel-shaped MLP encoder.
The model captures the semantic structure of the latent space and outperforms baseline regression methods, achieving moderate categorical accuracy in reconstructed images. However, performance remains below the original MindEye benchmarks, with noticeable spatial inaccuracies likely due to the pixel-agnostic nature of CLIP embeddings. The full pipeline’s VAE component for low-level structural reconstruction was not included due to resource limitations.
